# リバーシ（WebAssembly AI搭載）

ブラウザで遊べるリバーシゲームです。AIエンジンをWebAssemblyで実装しており、強力なAIと対戦できます。PWA（Progressive Web App）に対応しているため、スマートフォンのホーム画面に追加してオフラインでも遊べます。

## 特徴

- **強力なAI**: Alpha-Beta探索（深さ7〜8）＋終盤完全読み（残り12手以下）
- **高速動作**: AIエンジンをAssemblyScript → WebAssemblyにコンパイル
- **PWA対応**: iPhoneのホーム画面に追加してオフラインプレイ可能
- **レスポンシブデザイン**: PC・スマートフォン・タブレット対応
- **アニメーション**: 石の配置・反転をアニメーションで表示

## 動作環境

- Windows: Chrome, Edge, Firefox
- Android: Chrome
- iPhone/iPad: Safari（iOS 16.4以降推奨）

## プレイ方法

### オンラインプレイ

GitHub Pagesでホスティングしている場合は、そのURLをブラウザで開くだけで遊べます。

### ローカルでの起動

WebAssemblyは `file://` では動作しないため、ローカルHTTPサーバーが必要です。

```bash
# 依存パッケージのインストール（初回のみ）
npm install

# 開発サーバーの起動（http://localhost:3000 で起動）
npm start
```

ブラウザで `http://localhost:3000` を開いてください。

### スマートフォンへのインストール（PWA）

iPhoneの場合:
1. Safariでゲームのページを開く
2. 画面下部の共有ボタン（四角に上矢印のアイコン）をタップ
3. 「ホーム画面に追加」を選択

Androidの場合:
1. Chromeでゲームのページを開く
2. アドレスバー右の「…」メニューをタップ
3. 「ホーム画面に追加」または「アプリをインストール」を選択

## AIの仕組み

### 盤面表現（ビットボード）

64マスの盤面を2つの64ビット整数（黒・白）で表現します。ビット演算により合法手生成や反転計算を高速に行います。

### 探索アルゴリズム

- **NegaMax + Alpha-Beta枝刈り**: 悪い手を早期に打ち切ることで探索効率を向上
- **反復深化**: 深さ4→6→8と段階的に探索を深める
- **Move Ordering（手順整列）**: 角など有望な手を先に探索してα-β効率を上げる
- **終盤完全読み**: 残り12手以下で勝敗を確定させる完全探索に切り替え

### 評価関数

手番プレイヤーから見たスコアを以下の要素で計算します:

| 評価項目 | 説明 |
|---------|------|
| 位置評価 | 角（+500）・X打ちマス（-300）など戦略的重み付け |
| 角の獲得 | 1角につき+800点（安定石の基点） |
| X打ち/C打ちペナルティ | 角が空のときに隣接マスに置くと減点 |
| モビリティ | 着手可能数の差（多い方が有利） |
| フロンティア | 空きマスに隣接する石が少ない方が安定 |
| 安定石 | 確定石（以後反転しない石）の数 |
| パリティ | 終盤、最後の手を取れる側が有利 |

ゲームの進行フェーズ（序盤・中盤・終盤）に応じて各要素の重みを動的に変更します。

## WASMの再ビルド方法

AIエンジンを修正した場合は以下でWASMを再ビルドします:

```bash
# リリースビルド（最適化あり）
npm run asbuild:release

# デバッグビルド
npm run asbuild:debug
```

ビルドには Node.js と npm パッケージのインストールが必要です。
